About this project

As part of my studies in Technical Communication, I conducted in-depth research and created a comprehensive documentation project exploring the impact of Generative AI on the field of technical writing. This work demonstrates both my ability to produce clear, structured documentation and my skills in using modern web technologies — Markdown, HTML, CSS, and XML — to present research in an engaging, accessible way.

Research focus: How can GenAI tools like ChatGPT, GPT-3, and DALL·E assist technical writers in improving efficiency, enhancing content quality, and streamlining documentation workflows — without compromising accuracy or authorial control?

What the project covers

  • Practical AI applications — specific use cases where Generative AI supports technical writers: content generation, language refinement, review assistance, and localisation support
  • Interactive examples — demonstrations of how AI tools perform on real technical writing scenarios
  • Research & analysis — a deep dive into pros, cons, and ethical considerations of incorporating AI into the TW process
  • Workflow integration — practical guidance on where in the documentation workflow AI adds value and where human judgment remains essential

Technologies used

  • Markdown — structured, readable content creation
  • HTML, CSS, Just The Docs — web formatting and styling, clean user-friendly layout
  • XML — organising and structuring technical data for extensibility
  • GitHub Pages — transparent, accessible publishing with full version history

Key skills demonstrated

  • Research documentation — transforming in-depth research into structured, navigable documentation
  • AI literacy for TW — practical, critical understanding of GenAI capabilities and limitations in documentation contexts
  • Web publishing — end-to-end: from Markdown source to published GitHub Pages site
  • Content structuring — clear hierarchy, consistent terminology, audience-focused organisation
